# Images

Place actual photos of your build here:

- `hardware_setup.jpg` — photo of sensors wired to the Raspberry Pi
- `prototype.jpg` — photo of the assembled weather station enclosure
- `dashboard.png` — screenshot of any dashboard/visualization you build

These are intentionally left empty in this scaffold since they require real
photos of your physical hardware build. Drop your images in this folder with
the exact filenames above and they'll be picked up by the main README.
